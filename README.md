# AI Projects

A shared repository for cross-platform AI configuration, reusable skills, and per-project resources.

## Structure

| Folder | Purpose |
|--------|---------|
| `root/` | Core markdown files shared across all AI platforms |
| `skills/` | Reusable prompt skills and instructions |
| `projects/` | Per-project briefs and resources |

## Usage

- Start any new AI session by referencing the relevant files in `root/`
- Pull in skills from `skills/` as needed per task
- Add a new folder under `projects/` for each initiative, using `projects/_template/` as a starter
