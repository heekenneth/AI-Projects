# Coding Skill

Reusable instructions for AI coding tasks.

## When to Use

Writing, reviewing, debugging, or refactoring code.

## Instructions

- Prefer simple, readable solutions over clever ones
- No comments unless the WHY is non-obvious
- Write tests for non-trivial logic
- Flag security concerns immediately
- Ask before making changes beyond the stated scope

## Output Format

Code block → Brief explanation of key decisions (if needed)
