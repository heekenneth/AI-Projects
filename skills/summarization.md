# Summarization Skill

Reusable instructions for AI summarization tasks.

## When to Use

Condensing long documents, meeting notes, articles, or threads.

## Instructions

- Lead with the single most important takeaway
- Use bullet points for multiple findings
- Preserve key numbers, names, and dates
- Note anything requiring follow-up action
- Keep summaries to 20% of original length unless told otherwise

## Output Format

TL;DR (1-2 sentences) → Key Points → Action Items (if any)
